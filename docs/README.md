# docs/ — Reference Documentation

Folder ini untuk reference docs yang **sifatnya procedural, technical, atau panjang** — tidak cocok di MEMORY.md.

---

## Kapan Pakai Masing-Masing Folder

| Folder | Isi | Kapan Dipakai |
|--------|-----|---------------|
| `docs/` | Prosedur, how-to, technical specs, framework reference | Konten terlalu panjang/teknis untuk MEMORY.md |
| `knowledge/` | Riset eksternal, artikel, AI/tech trends | Wayan share link → auto-dump & summarize |
| `contexts/` | Project-specific context (brief notes per project) | Sebelum mengerjakan project spesifik |
| `tasks/` | lessons.md (apa yang sudah dipelajari), todo tracking | Setelah correction atau task selesai |
| `memory/` | Daily notes (YYYY-MM-DD.md) | Raw log tiap sesi |

---

| File | Deskripsi |

## Rule

- MEMORY.md hanya boleh punya **pointer** ke sini, bukan konten panjang
- Setiap file baru di docs/ → update tabel di atas
- Kalau udah tidak relevan → archive ke `docs/archive/` atau hapus

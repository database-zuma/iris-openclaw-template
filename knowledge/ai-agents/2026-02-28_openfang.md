## Apa Ini

Full Agent Operating System dibangun dari scratch dalam Rust. Bukan chatbot framework, bukan Python wrapper LLM. Single binary (~32MB), autonomous agents yang kerja 24/7 tanpa perlu diprompt user.

**Core concept — "Hands"**: pre-built capability packages yang jalan otomatis on schedule.

---

## 7 Bundled Hands

| Hand | Fungsi |
|------|--------|
| **Clip** | YouTube URL → vertical shorts + captions + thumbnails → Telegram/WA |
| **Lead** | Daily prospect discovery, enrich, score 0-100, deliver CSV/JSON |
| **Collector** | OSINT monitoring, knowledge graph, alert on changes (company/person/topic) |
| **Predictor** | Superforecasting, Brier scores, contrarian mode |
| **Researcher** | Deep research, CRAAP criteria, cited APA reports, multi-language |
| **Twitter** | Autonomous X manager, 7 content formats, approval queue |
| **Browser** | Web automation via Playwright, **mandatory purchase approval gate** |

## Quick Start

```bash
curl -fsSL https://openfang.sh/install | sh
openfang init && openfang start
# Dashboard: http://localhost:4200

openfang hand activate researcher  # starts working immediately
openfang migrate --from openclaw   # migrate semua agents/memory/skills
```

## Key Points

- **Massive curated list** of freely available programming books, courses, podcasts, cheat sheets, interactive tutorials, and playgrounds
- **45+ languages** including Bahasa Indonesia — both books (`books/free-programming-books-id.md`) and courses (`courses/free-courses-id.md`)
- **6 resource categories:**
  - 📚 **Books** — by programming language + by subject (2 English lists, 44 language-specific lists)
  - 🎓 **Courses** — free online courses (40+ language lists)
  - 📋 **Cheat Sheets** — all languages in one file
  - 🎮 **Interactive Tutorials** — hands-on coding (5 languages)
  - 🎙️ **Podcasts & Screencasts** — audio/video learning (20+ languages)
  - 🛝 **Playgrounds** — browser-based code editors (3 languages)
- **Searchable website:** https://ebookfoundation.github.io/free-programming-books-search/
- Community-driven: originated from a StackOverflow thread, moved to GitHub, now the largest open educational resource for programming
